## Wave Walker

Wave Walker is a platform game inspired by the T-Rex game. Made in C and SDL3.

### Prerequisites

Install SDL3 development libraries:

```bash
# Arch
sudo pacman -S sdl3

# Ubuntu/Debian
sudo apt install libsdl3-dev

# Fedora
sudo dnf install SDL3-devel
```

### Build

```
make
```

### Run

```
./build/game
```
